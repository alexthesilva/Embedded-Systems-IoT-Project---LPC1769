\doxysubsection{Detailed Description}
This package provides the interface for driving 4-\/bit H\+D44780-\/based L\+C\+Ds used in L\+P\+C\+Xpresso development board based on L\+P\+C1769 from N\+XP. 

The L\+CD data bits are connect to the microcontroller L\+P\+C1769 as show in the follow table\+: \tabulinesep=1mm
\begin{longtabu}spread 0pt [c]{*{2}{|X[-1]}|}
\hline
\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ L\+CD  }&\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ L\+P\+C1769   }\\\cline{1-2}
\endfirsthead
\hline
\endfoot
\hline
\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ L\+CD  }&\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ L\+P\+C1769   }\\\cline{1-2}
\endhead
\PBS\centering D0 .. D3  &\PBS\centering Not connected   \\\cline{1-2}
\PBS\centering D4 .. D7  &\PBS\centering P2.\+0 .. P2.\+3   \\\cline{1-2}
\PBS\centering EN  &\PBS\centering P0.\+10   \\\cline{1-2}
\PBS\centering RS  &\PBS\centering P0.\+11   \\\cline{1-2}
\PBS\centering WR  &\PBS\centering G\+ND   \\\cline{1-2}
\end{longtabu}
