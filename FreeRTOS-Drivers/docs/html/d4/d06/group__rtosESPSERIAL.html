Detailed Description

This package provides the interface for the ESP8266 with serial interface in a multi-threaded manner, along with AT command related functions.

Enumeration Type Documentation

◆ ESPState_t

enum ESPState_t

ESP8266 current state structure definition.

Enumerator
START 

Starts waiting for a desired outcome.

QUEUE 

Execution state reached the implemented queue. Only used internally

WAIT 

Waiting for a desired outcome.

FAIL 

Desired outcome has not been achieved.

SUCCESS 

Desired outcome has been achieved.
