\doxysubsection{Detailed Description}
This package provides the interface for driving the rotary and the push button. 

The rotary data bits are connected to the L\+P\+C1769 microcontroller as shown in the following table\+: \tabulinesep=1mm
\begin{longtabu}spread 0pt [c]{*{2}{|X[-1]}|}
\hline
\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ Rotary Button  }&\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ L\+P\+C1769   }\\\cline{1-2}
\endfirsthead
\hline
\endfoot
\hline
\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ Rotary Button  }&\PBS\centering \cellcolor{\tableheadbgcolor}\textbf{ L\+P\+C1769   }\\\cline{1-2}
\endhead
\PBS\centering C\+LK  &\PBS\centering P0.\+3   \\\cline{1-2}
\PBS\centering DT  &\PBS\centering P0.\+2   \\\cline{1-2}
\PBS\centering SW  &\PBS\centering P2.\+13   \\\cline{1-2}
\PBS\centering +  &\PBS\centering 3V3   \\\cline{1-2}
\end{longtabu}


The driver has the following behavior\+: i) two steps to vary in the same direction. i) one step when an inversion is performed.

After initialization clockwise direction is assumed.
